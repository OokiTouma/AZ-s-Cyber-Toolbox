## 功能特性

### 核心功能
1. **完整的角色创建流程**
   - 基础信息（姓名、等级、经验值）
   - 种族选择（含变体）
   - 子种族选择
   - 职业选择
   - 子职业选择（根据等级自动解锁）
   - 背景选择

2. **属性系统**
   - 六维属性（力量、敏捷、体质、智力、感知、魅力）
   - 种族属性加值自动计算
   - 属性掷骰生成（4d6去最低）
   - 属性调整值自动计算

3. **技能系统**
   - 18个技能
   - 熟练技能选择
   - 技能加值自动计算

4. **专长系统**
   - 67个专长
   - 先决条件自动检查
   - 专长效果详细描述

5. **法术系统**
   - 按职业筛选法术
   - 按等级分类
   - 法术详细信息（学派、施法时间、范围等）

6. **兼职系统**
   - 支持多职业兼职
   - 兼职等级管理
   - 自动计算总等级

7. **角色卡预览**
   - 实时更新
   - 完整的属性展示
   - 特性与能力详细描述
   - 技能列表
   - 法术列表

8. **数据管理**
   - 保存角色到本地存储
   - 加载已保存角色
   - 导出为PDF（打印功能）

## 使用说明

1. 打开 `index.html` 文件
2. 按照界面提示逐步创建角色
3. 选择种族后，如果有变体或子种族会自动显示
4. 选择职业后，达到相应等级时会解锁子职业选择
5. 添加专长时会自动检查先决条件
6. 施法职业可以从法术列表中选择法术
7. 可以随时保存角色或导出为PDF
