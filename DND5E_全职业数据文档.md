## 一、基础职业（玩家手册）

### 1. 野蛮人 Barbarian

**核心特质**
| 项目 | 内容 |
|------|------|
| 主要属性 | 力量 |
| 生命值骰 | d12 |
| 豁免熟练 | 力量、体质 |
| 技能熟练 | 选择2项：驯兽、运动、威吓、自然、察觉、生存 |
| 武器熟练 | 简易武器、军用武器 |
| 护甲熟练 | 轻甲、中甲、盾牌 |

**职业特性**

| 等级 | 特性名称 | 特性简介 |
|------|----------|----------|
| 1级 | **狂暴(Rage)** | 进入狂暴状态，获得力量检定和豁免优势，近战武器伤害+2，获得钝击、穿刺、挥砍伤害抗性。持续1分钟，需受到伤害或进行攻击维持。 |
| 1级 | **无甲防御(Unarmored Defense)** | 未着甲且持盾时，AC = 10 + 敏捷调整值 + 体质调整值。 |
| 2级 | **危险感知(Danger Sense)** | 对可见的陷阱、法术等危险获得敏捷豁免优势。 |
| 2级 | **鲁莽攻击(Reckless Attack)** | 进行近战武器攻击时，本回合获得攻击检定优势，但对你进行的攻击检定也获得优势。 |
| 3级 | **选择道途(Primal Path)** | 选择一条道途，获得特定的野蛮人子职业能力。 |
| 5级 | **额外攻击(Extra Attack)** | 攻击动作时可进行两次攻击而非一次。 |
| 5级 | **快速移动(Fast Movement)** | 未着重甲时，速度+10尺。 |
| 7级 | **野性本能(Feral Instinct)** | 先攻检定具有优势；无法 surprised 时无法行动。 |
| 9级 | **残暴暴击(Brutal Critical)** | 狂暴时暴击额外增加1个伤害骰（1d6）。 |
| 11级 | **坚韧狂暴(Relentless Rage)** | 狂暴时若降至0HP但未被直接杀死，可进行DC10的体质豁免，成功则改为降至1HP。每次使用DC+5。 |
| 15级 | **持久狂暴(Persistent Rage)** | 狂暴现在只需在回合开始时选择是否维持，不会因未攻击或未受伤害而提前结束。 |
| 18级 | **不屈勇武(Indomitable Might)** | 若力量检定结果低于力量值，可直接使用力量值代替。 |
| 20级 | **原初斗士(Primal Champion)** | 力量和体质+4（上限24）；狂暴时获得+4AC。 |

**子职业（道途）**

#### 1.1 狂战士道途 Path of the Berserker

| 等级 | 特性名称 | 特性简介 |
|------|----------|----------|
| 3级 | **狂热(Frenzy)** | 狂暴时可选择进入狂热状态，每回合可用附赠动作进行近战武器攻击，但获得一级力竭。 |
| 6级 | **心智怒狂(Mindless Rage)** | 狂暴时免疫魅惑和恐慌。 |
| 10级 | **威吓存在(Intimidating Presence)** | 可尝试威吓一个生物，失败则陷入恐慌直至你的下一回合结束。 |
| 14级 | **反击(Retaliation)** | 当5尺内的生物对你造成伤害时，可用反应进行近战武器攻击。 |

#### 1.2 图腾武者道途 Path of the Totem Warrior

| 等级 | 特性名称 | 特性简介 |
|------|----------|----------|
| 3级 | **兽魂行者(Spirit Seeker)** | 获得野兽交谈和探知动植物法术。 |
| 3级 | **图腾兽(Totem Spirit)** | 选择一种图腾兽（熊、鹰、狼、虎等），获得对应特性：熊-狂暴时除心灵外全伤害抗性；鹰-借机攻击劣势；狼-友军5尺内攻击生物时其首次攻击优势。 |
| 6级 | **兽魂行者特性** | 根据选择的图腾兽获得额外能力：熊-负重翻倍；鹰-黑暗视觉、侦察检定优势；狼-追踪检定优势。 |
| 14级 | **图腾之力(Totemic Attunement)** | 获得更强大的图腾能力：熊-5尺内友军获得你的伤害抗性；鹰-飞行速度等于步行速度；狼-击倒生物后可进行额外攻击。 |

#### 1.3 先祖守卫道途 Path of the Ancestral Guardian（珊娜萨的万事指南）

| 等级 | 特性名称 | 特性简介 |
|------|----------|----------|
| 3级 | **先祖战士(Ancestral Protectors)** | 狂暴时攻击的第一个目标对你的攻击具有劣势，且其攻击除你之外的生物时目标具有抗性。 |
| 6级 | **先祖护佑(Spirit Shield)** | 可用反应减少10尺内友军受到的伤害（2d6），随等级提升。 |
| 10级 | **咨询灵魂(Consult the Spirits)** | 短休时可进行仪式，获得占卜或死者交谈的效果。 |
| 14级 | **复仇先祖(Vengeful Ancestors)** | 精神护盾现在可将减少的伤害反弹给攻击者。 |

#### 1.4 风暴先驱道途 Path of the Storm Herald（珊娜萨的万事指南）

| 等级 | 特性名称 | 特性简介 |
|------|----------|----------|
| 3级 | **风暴光环(Storm Aura)** | 狂暴时在10尺光环内产生效果：沙漠-火焰伤害；海洋-闪电伤害；苔原-寒冷伤害。 |
| 6级 | **风暴之魂(Storm Soul)** | 获得环境抗性：沙漠-火焰抗性；海洋-闪电抗性；苔原-寒冷抗性。 |
| 10级 | **风暴之怒(Shielding Storm)** | 光环内的友军也获得你的风暴之魂抗性。 |
| 14级 | **狂暴风暴(Raging Storm)** | 风暴光环增强：沙漠-击退；海洋-反应击倒；苔原-冻结。 |

#### 1.5 狂热者道途 Path of the Zealot（珊娜萨的万事指南）

| 等级 | 特性名称 | 特性简介 |
|------|----------|----------|
| 3级 | **神圣狂怒(Divine Fury)** | 狂暴时首次命中生物时额外造成1d6+一半野蛮人等级的光耀或黯蚀伤害。 |
| 3级 | **战士之魂(Warrior of the Gods)** | 复活你所需的法术无需材料成分。 |
| 6级 | **狂热焦点(Fanatical Focus)** | 狂暴时若豁免失败，可用反应重骰。 |
| 14级 | **狂怒之魂(Zealous Presence)** | 可让10尺内生物在1分钟内攻击检定和豁免获得优势。 |

#### 1.6 巨人道途 Path of the Giant（巨人之荣耀）

| 等级 | 特性名称 | 特性简介 |
|------|----------|----------|
| 3级 | **巨人之力(Giant Power)** | 学会一门巨人语，获得德鲁伊伎俩戏法。 |
| 3级 | **巨人之躯(Giant's Rage)** | 狂暴时体型增大一级，触及+5尺，力量检定和豁免获得优势。 |
| 6级 | **元素之怒(Elemental Cleaver)** | 可将武器注入元素力量，额外造成1d6元素伤害。 |
| 10级 | ** mighty impel** | 可将生物或物体投掷出去，造成伤害。 |
| 14级 | **泰坦之力(Demigod's Rage)** | 狂暴时体型可增至巨型，触及再+5尺，元素伤害增至2d6。 |

### 2. 吟游诗人 Bard

**核心特质**
| 项目 | 内容 |
|------|------|
| 主要属性 | 魅力 |
| 生命值骰 | d8 |
| 豁免熟练 | 敏捷、魅力 |
| 技能熟练 | 任意3项 |
| 武器熟练 | 简易武器、手弩、长剑、刺剑、短剑 |
| 护甲熟练 | 轻甲 |

**职业特性**

| 等级 | 特性名称 | 特性简介 |
|------|----------|----------|
| 1级 | **施法(Spellcasting)** | 使用魅力施法，法术位为完整施法者进度，可施展吟游诗人法术列表中的法术。 |
| 1级 | **吟游激励(Bardic Inspiration, d6)** | 短休后恢复，可用反应给予60尺内生物一个d6，用于攻击、豁免或能力检定，10分钟内可用。 |
| 2级 | **万事通(Jack of All Trades)** | 无熟练的技能检定获得半数熟练加值（向下取整）。 |
| 2级 | **休憩曲(Song of Rest, d6)** | 短休时演奏音乐，恢复生命值的生物额外恢复1d6生命值。 |
| 3级 | **选择学院(Bard College)** | 选择一个吟游诗人学院，获得特定的子职业能力。 |
| 3级 | **专精(Expertise)** | 选择2项已熟练的技能或工具，熟练加值翻倍。 |
| 5级 | **吟游激励提升(d8)** | 激励骰提升至d8。 |
| 5级 | **激励之源(Font of Inspiration)** | 短休或长休后恢复所有吟游激励次数。 |
| 6级 | **反迷惑(Countercharm)** | 动作开始演奏，10尺内友军对抗魅惑/恐慌豁免优势。 |
| 6级 | **学院特性** | 获得所选学院的特殊能力。 |
| 9级 | **吟游激励提升(d10)** | 激励骰提升至d10。 |
| 10级 | **专精(第二项)** | 再选择2项技能或工具获得专精。 |
| 10级 | **万事通提升** | 万事通现在适用于先攻检定。 |
| 13级 | **吟游激励提升(d12)** | 激励骰提升至d12。 |
| 14级 | **学院特性** | 获得所选学院的额外能力。 |
| 14级 | **魔法奥秘(Magical Secrets, 4个)** | 从任意职业法术列表中选择4个法术加入已知法术。 |
| 15级 | **激励之源提升** | 长休后恢复所有激励次数，短休后恢复1个。 |
| 18级 | **魔法奥秘(2个)** | 再从任意职业法术列表中选择2个法术。 |
| 20级 | **卓越灵感(Superior Inspiration)** | 先攻检定开始时若无可用的激励次数，获得1个。 |

**子职业（学院）**

### 3. 牧师 Cleric

**核心特质**
| 项目 | 内容 |
|------|------|
| 主要属性 | 感知 |
| 生命值骰 | d8 |
| 豁免熟练 | 感知、魅力 |
| 技能熟练 | 选择2项：历史、洞悉、医药、说服、宗教 |
| 武器熟练 | 简易武器 |
| 护甲熟练 | 轻甲、中甲、盾牌 |

**职业特性**

| 等级 | 特性名称 | 特性简介 |
|------|----------|----------|
| 1级 | **施法(Spellcasting)** | 使用感知施法，法术位为完整施法者进度，可施展牧师法术列表中的法术，可准备法术。 |
| 1级 | **神圣领域(Divine Domain)** | 选择一个神圣领域，获得领域法术和特性。 |
| 2级 | **引导神力(Channel Divinity, 1次/短休)** | 短休后恢复，获得领域特定的引导神力选项。 |
| 5级 | **摧毁不死生物(Destroy Undead, CR 1/2)** | 引导神力驱散不死生物时，CR 1/2或更低的不死生物直接被摧毁。 |
| 6级 | **引导神力(2次)** | 每短休可使用2次引导神力。 |
| 6级 | **领域特性** | 获得所选领域的额外能力。 |
| 8级 | **摧毁不死生物(CR 1)** | 提升至CR 1。 |
| 8级 | **领域特性** | 获得神圣打击或神圣之箭（视领域而定）。 |
| 10级 | **神圣干预(Divine Intervention)** | 可请求神祇直接干预，成功率等于牧师等级百分比，成功后7天内无法再次使用。 |
| 11级 | **摧毁不死生物(CR 2)** | 提升至CR 2。 |
| 14级 | **摧毁不死生物(CR 3)** | 提升至CR 3。 |
| 17级 | **摧毁不死生物(CR 4)** | 提升至CR 4。 |
| 17级 | **领域特性** | 获得领域的最高级能力。 |
| 20级 | **神圣干预提升** | 神圣干预自动成功。 |

#### 3.2 生命领域 Life Domain

| 等级 | 特性名称 | 特性简介 |
|------|----------|----------|
| 1级 | **领域法术** | 获得祝福术、治愈真言等法术。 |
| 1级 | **生命门徒(Disciple of Life)** | 使用1环或更高环阶法术恢复生命值时，目标额外恢复2+法术环阶点生命值。 |

#### 3.3 光明领域 Light Domain

| 等级 | 特性名称 | 特性简介 |
|------|----------|----------|
| 1级 | **领域法术** | 获得燃烧之手、妖火术等法术。 |
| 1级 | **警示之光(Warding Flare)** | 反应使攻击你的生物攻击检定具有劣势。 |
